Release Notes

Release notes for Liquibase Open Source and Liquibase Pro.

Liquibase 4.26.0 – February 6, 2024

Liquibase 4.26.0 contains several Notable Changes for Liquibase Pro users: if conditionals, Simpler PatternAFollowedByPatternB, and Checks Run Report.

For more information, see Liquibase 4.26.0 Release Notes.

Liquibase 4.25.1 – December 18, 2023

Liquibase 4.25.1 includes improvements to Observability like Update Report and Drift Report enhancements, the new connect command, pro-global-end-delimiter parameter, and Structured Logging enhancements.

For more information, see Liquibase 4.25.1 Release Notes.

Liquibase 4.25.0 – November 13, 2023

Liquibase 4.25.0 includes new features like the database Drift Report, the ability to organize your database checks in Quality Checks Packages, and enhancements to Structured Logging functionality.

For more information, see Liquibase 4.25.0 Release Notes.

Note: In one Liquibase 4.25.0 file, the <version> was set as "release-SNAPSHOT" when it should have been "4.25.0". This file, liquibase-core-4.25.0.jar/pom.xml, has been updated. This text-only change contains no code or functional impact, but does result in a different checksum.

LiquibaseExtensions: AWS S3 1.1.3, Hashicorp Vault 2.0.4, AWS Secrets Manager 1.0.4 – October 25, 2023

The following Pro extensions have been updated to ensure compatibility with more recent versions of Liquibase Pro. These extensions are compatible with Liquibase versions 4.19.0 through 4.24.0. Learn more here: 

  • AWS S3 extension v1.1.3
  • Hashicorp Vault extension v2.0.4
  • AWS Secrets Manager extension v1.0.4

The AWS S3 extension has improved usability for users with large S3 buckets (over 1000 items).

Liquibase MongoDB Pro 1.3.0 – October 11, 2023

MongoDB users can now manage their existing mongosh scripts with Liquibase by adding comments containing Liquibase directives. This makes it easy to bring the power of Liquibase change management into an existing project team using MongoDB.

For more information, see Liquibase MongoDB Pro 1.3.0 Release Notes.

Liquibase 4.22.0 – May 11, 2023

Note: We have removed the 4.22.0 release of Liquibase. This version contains an updated algorithm for determining which changes Liquibase needs to deploy to a database. In some situations, the conversion to this new algorithm could result in eligible changes not being deployed to a target database.

Specifically, currently undeployed changesets with the runOnChange=true attribute may not be deployed if they were present in the first execution of Liquibase 4.22.0 against a given target database. If you have not upgraded to Liquibase 4.22.0, you are not impacted by this issue, or if you do not rely on runOnChange changesets which have changed before your upgrade.

To remedy this issue, run the clear-checksums command, then revert to Liquibase 4.21.1, and then run the changelog-sync command to recalculate the v8 checksums.

Liquibase v4.22.0 is a major release.

Learn more here: Liquibase 4.22.0 Release Notes
